# Retail Inventory Analytics Using Google Colab

This project analyzes retail inventory data using Python and Google Colab.

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

- Inventory Analysis

- Data Visualization

- Line Graph
